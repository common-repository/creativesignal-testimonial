CreativeSignal Testimonial allows you to present testimonials on your website.

== Description ==

CreativeSignal Testimonial allows you to present customer testimonials on your website in a sidebar through CreativeSignal Testimonials Widget or in a page or post through the short-code.

Features:

<ol>
<li>Add testimonials to your website.</li>
<li>Simple interface to enter and edit testimonials.</li>
<li>Display testimonials through Sidebar (Widget) or Short-code.</li>
<li>Fade-in and fade-out animation with short-code.</li>
<li>Scroll animation with sidebar (Widget).</li>
</ol>

PS: For future reference how to use the plugin, check out our documentation here: http://creative-signal.com/wordpress/wordpress-plugins/creativesignal-testimonials

== Installation ==

Setting up CreativeSignal Testimonials is pretty easy, just follow instructions below:

<ol>
<li>Download the installation package.</li>
<li>In the WordPress back-end, go to Plugins > Add New  and select Upload.</li>
<li>Install and active the plug-in.</li>
<li>Go to CS testimonials menu item.</li>
<li>Add one or more testimonials to your website.</li>
<li>Go in Appearance > Widgets and add the CreativeSignal Testimonials Widget to one of your sidebars. Adjust options.</li>
<li>Paste [cs-testimonials] in some of your pages/posts to display all testimonials.</li>
</ol>
